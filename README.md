# ✨Description ![evee](https://media.tenor.com/sj3IHPPDOxEAAAAi/eevee-eevee-evolution.gif)

Recreation of Pokémon Pinball for the Game Boy. This recreation is made on a surface level compared with the original one, meaning that many features are not included, as more than one map.

This recreation includes:

- Flippers and launcher for the ball
- Score system, with high score and last score
- Life system, which resets the game once you lost all the lifes
- Collision system with all the pokemons
- Debug features
- Repository URL: [REPOSITORY](https://github.com/abelnunez5/Pokemon_Pinball_2D)

# ✨Project Features ![Jolteon](https://media.tenor.com/0bIjwnKhaOcAAAAj/jolteon-pokemon-jolteon.gif)

- Game engine: Raylib, Box2D

- Programming language: C#, C++

- Visual style: Pixel art faithful to the original

- Physics and game mechanics inspired by the original game

- Game replicated from the GBC (Game Boy Color) version

- Sound and music based on the classic version

- Controls adapted for keyboard

 
# ✨Controls ![Espeon](https://media.tenor.com/cERJ7M_mH-cAAAAj/eeveelution-eevee-evolution.gif)

- **A/D** move Flippers
- **SPACEBAR** Charge ball
- **ENTER** Skyp intro
